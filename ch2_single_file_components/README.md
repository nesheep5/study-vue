# Single File Components

## Links
- https://vuejs.org/v2/guide/single-file-components.html
- https://cli.vuejs.org/guide/

## Proc
install 'vue-cli'.
```bash
npm install -g @vue/cli
```

create vue project.
```
vue create sfc-sample
```

run server.
```
cd sfc-sample
npm run serve
```

show browser.
```
http://localhost:8080
```
